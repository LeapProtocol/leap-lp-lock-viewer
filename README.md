# How to run  
## Pull this repo  
```
git clone https://github.com/LeapProtocol/leap-lp-lock-viewer.git  
cd leap-lp-lock-viewer
```

## Install dependencies  
```
pip install web3  
# or  
pip3 install web3
```
## Run the code  
```
python leap-lp-lock-viewer.py  
// OR  
python3 leap-lp-lock-viewer.py  
```
